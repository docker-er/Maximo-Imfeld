Descripcion del proyecto
  
Este proyecto propone el desarrollo de una pagina que brinde servicios de impresión 3D donde los usuarios puedan solicitar la fabricación de objetos basados en una idea u archivo, o comprar de nuestro catalogo existente.

Funciones:

Un chat para clientes que quieran solicitar un objeto especifico (que no esta en el catalogo).

Explorar un catálogo de diseños disponibles y comprar directamente objetos ya preparados para imprimir.

Un calculador de precios en caso de querer alterar/solicitar modelos.

Informacion pertinente del emprendimiento (Modelo de impresora, las dimensiones de los modelos catalogados, el tiempo de impresion, etc)

Design Inspirations and Creative Process
Design Inspirations

<p align="center"   >
Inspirations and creative process of the project:

Online maker platforms (such as 3D model marketplaces): These platforms inspired the idea of allowing users to browse and purchase ready-made models

On-demand service platforms: The workflow of uploading a request and receiving a service influenced the user interaction model

 Modern web design: Clean interfaces that help users quickly understand how to upload files, choose materials, and place orders.

These inspirations guided the design towards a simple, functional, and accessible platform
